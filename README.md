## Projeto

Desenvolvimento do site **Portfólio Lobo** seguindo o projeto prático do curso de **HTML e CSS para Iniciantes** da [Origamid](https://www.origamid.com). O objetivo foi colocar em prática os fundamentos de estruturação semântica com HTML5 e estilização com CSS3, construindo uma página de portfólio completa do zero.

> 🌐 [Acesse o site](https://lucas-m3nezes.github.io/portfolio-origamid/) &nbsp;|&nbsp; 📁 [Acesse o repositório](https://github.com/Lucas-m3nezes/portfolio-origamid)

## O que aprendi

- **Flexbox** — organização e alinhamento de elementos em uma dimensão
- **CSS Grid** — construção de layouts bidimensionais
- **Responsividade** — adaptação do layout para diferentes tamanhos de tela
- **Semântica HTML** — uso correto de tags como `<header>`, `<main>`, `<section>`, `<footer>` e outras
